Usage & Principles

Always

Use noise/static textures as backgrounds. Let symbols breathe in dark space. Keep the pink hot and singular ‚Äî it's the signal in the void. Treat every element like it was found scratched into a wall or intercepted from a dead channel. Embrace asymmetry. Let faces emerge from the composition itself.

Never

Use clean gradients or polished glass effects. No drop shadows that look "material." No rounded-corner card UI. No stock photography. No light mode. Never center-align body text. Never use more than one accent color per composition. The aesthetic is found artifact, not designed product.

Texture Rules

Layer SVG noise at 3-5% opacity over everything. Use scanlines on dark surfaces. Dashed borders over solid when possible. Monospace for all UI chrome. Cross-hatching and stipple patterns for fills. Everything should feel slightly degraded, like a photocopy of a photocopy of a ritual diagram.

Motion

Slow, breathing glows (3-5s cycles). No bouncy spring animations. Subtle flicker on state changes. Elements should feel like they're being revealed by candlelight, not animated by a UI library. Use opacity transitions over transforms. Static bursts for attention.
